# Sanjay P N - Portfolio Website

A modern, minimalist portfolio showcasing my work as a Machine Learning Engineer, AI Developer, and Full-Stack Developer.

## About

This portfolio highlights my expertise in AI/ML development, full-stack web applications, and generative AI systems. Built with React and featuring smooth animations, responsive design, and a clean professional aesthetic.

## Tech Stack

- React 18
- React Router DOM
- Framer Motion (animations)
- CSS3 with modern layouts
- Create React App

## Features

- Animated typing effect cycling through roles
- Smooth scroll animations and transitions
- Responsive design for all devices
- Project showcase section
- Skills and experience sections
- Professional minimalist design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Deployment

This site is deployed on Vercel. Any push to the main branch automatically triggers a new deployment.
